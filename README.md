Introduction

Ce projet a pour objectif de mettre en place un Service Web SOAP en utilisant la technologie JAX-WS en Java. Le service est exécuté localement et permet à des clients d’invoquer différentes méthodes à distance via le protocole SOAP. Ce projet permet de comprendre les bases du fonctionnement des services Web, notamment la description via le fichier WSDL et les échanges de données XML entre le client et le serveur.

Technologies utilisées

Java 8 ou version supérieure

JAX-WS : Java API for XML Web Services

JAXB : Java Architecture for XML Binding pour la sérialisation des objets en XML

SOAP : Simple Object Access Protocol pour les échanges de données

HTTP : Protocole de communication

IntelliJ IDEA : IDE pour le développement
